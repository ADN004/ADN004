### Telegram Bot Suite — Production SaaS

> Private repos · Live with active daily users

Self-hosted suite of automation bots running 24/7 on Ubuntu VPS:

- **FileStoreBot** — file indexing, storage, and on-demand delivery backed by PostgreSQL + Redis caching
- **AutoForward Bot** — channel-to-channel forwarding with **tiered subscription plans** and isolated per-pair configs
- **File Gate Bot** — access control via **AI-assisted screenshot verification** (Groq / Gemini vision models)

Operational layer: Docker Compose orchestration, automated compressed `pg_dump` backups delivered to Telegram, scheduled tasks via APScheduler.

`Python` `aiogram 3` `Pyrogram` `PostgreSQL` `Redis` `Docker`
